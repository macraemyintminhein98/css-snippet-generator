# CSS Snippet Generator

A simple web application to generate common CSS snippets like box-shadows, linear gradients, and button styles based on user input. Get instant previews and copy-pastable code for your projects.

## Features:
- Generates CSS for Box Shadows, Linear Gradients, and Button Styles.
- Live visual preview of the generated CSS.
- Easy-to-use input controls (sliders, color pickers).
- 'Copy to Clipboard' functionality.

## How to Use:
1. Select the type of CSS snippet you want to generate.
2. Adjust the parameters using the provided controls.
3. See the live preview update instantly.
4. Copy the generated CSS code to your clipboard and use it in your projects.

## Technologies:
- HTML
- Tailwind CSS (for styling)
- Vanilla JavaScript (for logic)
